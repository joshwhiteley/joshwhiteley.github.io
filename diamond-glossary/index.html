XGBoost

An optimized gradient boosting library designed to improve speed and performance of machine learning models.

Modeling Terms

Overfitting

When a model learns the training data too well, capturing noise and yielding poor performance on new data.

Cross-Validation

A technique for evaluating model performance by partitioning data into subsets for training and validation.

Metrics

Accuracy

The proportion of true results (both true positives and true negatives) among the total number of cases examined.

Precision

The proportion of true positive results in relation to the total positive predictions made by the model.
